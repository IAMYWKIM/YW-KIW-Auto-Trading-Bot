## [3.2.0] — 2026-05-20

### ⚡ 단타 스캐너 전면 개편 — VWAP 제거 + 순수 모멘텀 5신호

#### 핵심 변경 — `scanner.py` v1.x → v2.0

**VWAP 완전 제거**

기존 문제: VWAP은 장중 누적 평균이라 고점 이후 항상 현재가 < VWAP → 눌림 매매 자체가 구조적으로 불가능했음 (코스모로보틱스·주성엔지니어링 미포착 원인)

**새 점수 체계 (최대 100점 + Fib 15점)**

| 신호 | 배점 | 기준 |
|------|------|------|
| 거래대금 폭발 | 30점 | 200억+ =30 / 100억+ =25 / 50억+ =18 |
| 거래량 폭발 | 25점 | 10배+ =25 / 5배+ =20 / 3배+ =13 |
| 가격 모멘텀 | 20점 | 10~15% =20 / 7~10% =16 / 5~7% =12 |
| 당일 Fib 눌림 | 15점 | 고점 대비 0.236~0.382 황금구간 +15점 |
| 시간대 | 10점 | 09:00~09:30 =10 / 09:30~10:00 =8 |

**환기종목 자동 제외**
- 관리종목 / 투자주의 / 거래정지예고 종목 장전 캐시 + 실시간 2차 체크

## [3.0.0] — 2026-05-18

### 🆕 전략 C 추가 — 상한가 선진입 (25% 구간 복합 신호 매수)

#### 신규 파일

| 파일 | 설명 |
|------|------|
| `position_registry.py` | 3전략 공유 포지션 레지스트리 (thread-safe, 충돌·자금 경합 방지) |
| `limit_config.py` | 전략C 전용 설정 관리 (`data/limit_config.json` 영속) |
| `limit_scanner.py` | 25%+ 종목 실시간 스캐너 (기존 broker API 재사용) |
| `strategy_limit.py` | 7대 신호 복합 스코어링 + 진입/청산 로직 |
| `INTEGRATION_GUIDE.md` | main.py 패치 단계별 가이드 |

#### 전략C 핵심 알고리즘

- **7대 복합 신호 스코어링 (0~100점)**
  - 테마 모멘텀 25 % — 선행 종목 상한가 비율
  - 뉴스 감성 20 % — 키워드 강도 차등 가중
  - 기관·외인 수급 20 % — 순매수 전환 감지
  - 거래량 패턴 15 % — 전일比 폭발 + 매수 비중
  - 기술적 지표 10 % — 이평선 정배열 / RSI / 잔량
  - 소셜 버즈 5 % (예정)
  - 시장 전체 흐름 5 %

- **테마 클러스터 DB 내장** — 2차전지, AI반도체, 방산, 바이오, 로봇, 원전

- **익일 갭 구간별 매도 자동화**
  - +5 % 이상 갭상승 → 동시호가 전량 매도
  - +2~5 % → 동시호가 50 % + 장초반 50 %
  - 보합 → 09:30 강제청산
  - 갭하락 -3 % → 즉시 손절

#### 공통 인프라 추가 (전략 A·B 에도 적용)

- **`PositionRegistry`** — 매수 전 중복·자금·블랙리스트 3중 체크
  - 전략별 자금 한도: A 25 % / B 30 % / C 30 % / 예비 10 %
  - 손절 후 30분 쿨다운 자동 적용
  - 전략 우선순위: C(3) > A(2) > B(1)

#### 스케줄 추가

| 시각 | 작업 |
|------|------|
| 08:50 | 전략C 장전 초기화 (거래량 캐시 + 포지션 복원) |
| 08:00~09:30 (매분) | 익일 아침 갭 매도 감시 |
| 09:00~14:30 (30초) | 장중 스캔·진입·청산 루프 |
| 09:25 | 강제청산 5분 전 경고 |
| 09:30 | 미청산 전량 강제청산 |
| 15:40 | 당일 결산 리포트 |

#### 텔레그램 명령어 추가

| 명령어 | 설명 |
