## 전략 A — 종가베팅

장 마감 직전(15:10~15:20) 다음날 상승 가능성이 높은 종목을 매수하고,
다음날 익절/손절하는 단기 전략입니다.

### 후보 선정 3단계

```
1단계: 후보 풀 수집 (3가지 소스)
   소스1: ka10016 — 신고가 종목
   소스2: ka10023 — 당일 거래량 급증
   소스3: ka10024 — 거래량 갱신

2단계: 핵심 필터
   ✅ 최근 5일 내 +5% 이상 급등일 존재
   ✅ 거래대금 100억 이상 (전일 기준)
   ✅ MA5 > MA20 정배열
   ✅ RSI 30~80 범위
   ✅ 20일 고점 대비 -20% 이내

3단계: 점수화 후 정렬
   거래대금(35점) + 급등강도(30점) + 수급(20점) + 신고가근접(15점)
```

### 매수 조건 (15:10~15:20)

```
전일 종가 대비 -0.5% ~ -10% 눌림 중인 종목
```

### 매도 전략 (D+1)

```
08:00~08:50  NXT 프리마켓 +2% 이상 → 즉시 매도
09:00~10:00  +3% 이상 → 50% 부분 매도
장중          +5% 이상 → 전량 매도
장중          -3% 이하 → 손절
장중          고점 대비 -2% → 트레일링 스탑
15:00         미청산 → 강제 청산
```

## 전략 B — 장중단타 (v3.2 개편)

09:00~14:00 장중 급등 종목을 당일 매수·매도하는 단타 전략입니다.

### v3.2 핵심 변경 — VWAP 제거 + 순수 모멘텀 5신호

기존 VWAP 필터가 고점 이후 모든 눌림 매매를 차단하는 구조적 문제를 해결했습니다.

### 5신호 모멘텀 점수 (최대 115점)

```
① 거래대금 폭발 (30점)
   200억+ =30 / 100억+ =25 / 50억+ =18 / 10억+ =10

② 거래량 폭발 (25점)
   10배+ =25 / 5배+ =20 / 3배+ =13 / 2배+ =8

③ 가격 모멘텀 (20점)
   10~15% =20 / 7~10% =16 / 5~7% =12 / 15~20% =10

④ 당일 고가 대비 Fib 눌림 (15점)  ★ 신규
   Fib 0.236~0.382 황금구간 =15 / 0.382~0.618 =10 / 0~0.236 =5

⑤ 시간대 가중 (10점)
   09:00~09:30 =10 / 09:30~10:00 =8 / 10:00~11:30 =5

보너스: 신규상장 60일 이내 +5점
```

### 환기종목 자동 제외 ★ 신규

장전 08:50 초기화 시 관리종목·투자주의·거래정지예고 종목을 캐시에 등록하여 스캔에서 자동 제외합니다.

### 진입 판단 구조

```
스캔 점수 (5신호) + Fib 보너스 ≥ min_entry_score(35점) → 진입
연속 손절 시: 전략 전체 멈춤 → 종목별 60분 쿨다운으로 변경
```

## 전략 C — 상한가선진입 ★ NEW

장중 25% 이상 급등 종목에 대해 7대 신호를 복합 분석하여 상한가 도달 전 선진입하고,
익일 아침 갭 구간에 따라 매도합니다.

### 7대 복합 신호 스코어링 (0~100점)

| 신호 | 가중치 | 주요 판단 기준 |
|------|--------|----------------|
| 테마 모멘텀 | 25 % | 동일 테마 선행 종목 상한가 비율 |
| 뉴스 감성 | 20 % | 공시·뉴스 키워드 강도 차등 |
| 기관·외인 수급 | 20 % | 순매수 전환 감지 |
| 거래량 패턴 | 15 % | 전일比 폭발 + 매수 비중 |
| 기술적 지표 | 10 % | 이평선 정배열 / RSI / 잔량 굳히기 |
| 소셜 버즈 | 5 % | 커뮤니티 관심도 (예정) |
| 시장 전체 흐름 | 5 % | KOSPI/KOSDAQ 방향성 |

### 진입 기준

```
복합점수 ≥ 70점 (기본값, 설정 변경 가능)
체결강도 ≥ 150%
매도잔량 / 매수잔량 ≤ 10%
테마 선행 종목 상한가 1개 이상 (설정으로 비활성화 가능)
14:30 이전 스캔
```

### 익일 매도 전략

```
갭상승 +5% 이상   → 동시호가 전량 매도
갭상승 +2~5%     → 동시호가 50% + 장초반 50%
보합 / 미확인     → 09:30 강제청산
갭하락 -3% 이하  → 즉시 손절
```

## 공통 인프라 — PositionRegistry (v3.0 신규)

3전략이 동일한 종목을 중복 매수하거나 자금을 경합하는 것을 방지합니다.

```
매수 전 3중 체크
  1. 블랙리스트 쿨다운 — 손절 후 30분 재매수 금지
  2. 중복 포지션 — 이미 보유 중인 종목 차단 (우선순위 낮은 전략)
  3. 자금 한도 — 전략별 한도 초과 차단

전략 우선순위: C(3) > A(2) > B(1)
자금 한도: A 25% / B 30% / C 30% / 예비 10%
```

## 버전 관리 규칙

```
X.Y.Z
│ │ └── Z: 버그 수정 / 소수 개선 (패치)
│ └──── Y: 기능 추가 / 전략 조건 변경 (마이너)
```

각 변경 시:
1. `VERSION` 파일 업데이트
2. `CHANGELOG.md` 에 내용 추가
3. 변경된 `.py` 파일 헤더의 버전 업데이트
4. `deploy_kiw` 로 배포
