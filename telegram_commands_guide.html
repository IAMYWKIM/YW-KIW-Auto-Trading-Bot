🆕 하이브리드 모드 — 수동 주도주 지정

유저가 직접 주도주를 지정 → 동일한 단타 로직으로 자동 매매
💡 하이브리드 모드란?
자동 스캐너는 상승률/거래대금/거래량으로 종목을 선정합니다.
하지만 장 초반 주도주·테마는 사람의 눈으로 더 정확히 판단할 수 있습니다.
유저가 직접 주도주를 지정하면, 스캐너 필터를 우회하고 동일한 단타 로직(손절/익절/트레일링)으로 자동 매매합니다.
1
/scalp_add 247540 입력 → 에코프로비엠 감시 시작
2
30초 루프마다 실시간 시세 조회 → 진입 조건 체크 → 자동 매수
3
손절/익절/트레일링 동일 청산 로직 적용 (자동 스캔 종목과 동일)
4
/scalp_remove 247540 입력 → 신규 진입 중단 (보유분 유지)
/scalp_add
주도주 수동 감시 추가 NEW
종목코드로 지정 → 30초 후부터 자동 매수 시도. 스캐너 필터 우회
▶
구문
/scalp_add 종목코드 [종목명]
예제
/scalp_add 247540에코프로비엠 — 이름 자동 조회
/scalp_add 005930 삼성전자이름 직접 지정
/scalp_add 000660SK하이닉스
응답 예시
✅ 하이브리드 감시 추가 📌 종목: 에코프로비엠(247540) 💰 현재가: 35,200원 이 종목은 다음 30초 루프부터 자동으로 매수 시도합니다. 진입 조건: 포지션 여유 + 현금 + 진입 마감 시각 이내 /scalp_remove 247540 — 감시 중단 /scalp_watchlist — 전체 목록 확인
🎯 스캐너 필터 우회: 상승률·거래대금·거래량 조건 없이 진입
진입 조건 유지: 포지션 수 / 현금 / 시간 마감 / 쿨다운은 동일 적용
알림 표시: 매수/매도 알림에 source: MANUAL로 자동·수동 구분
/scalp_remove
수동 감시 중단 NEW
신규 진입만 중단. 보유 중이면 포지션은 유지하고 기존 청산 로직 계속
▶
구문
/scalp_remove 종목코드
예제
/scalp_remove 247540에코프로비엠 감시 중단
/scalp_remove 005930삼성전자 감시 중단
응답 예시 (보유 중일 때)
⏸ 에코프로비엠(247540) 신규 진입 중단 현재 보유 포지션은 기존 청산 로직으로 계속 관리됩니다
⚠️ 감시 중단 후에도 이미 보유한 포지션은 손절/익절/트레일링이 그대로 적용됩니다. 즉시 청산이 필요하면 /scalp_exit_all을 사용하세요.
