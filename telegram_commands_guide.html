종가베팅 — 스캔 · 분석

후보 종목 스캔, 승률 분석, 결과 업데이트
/scan
종가베팅 후보 종목 수동 스캔
조건 A~G 필터로 내일 매수 후보를 스캔합니다. 결과는 자동으로 로그에 저장 (10~30초 소요)
▶
구문
/scan
응답 예시
[ 종가베팅 스캔 2개 ] 14:32 ⭐ 에코프로비엠(247540) 점수:85 [RISE_RANK] 🔥급등:+12.3%(D-1) 📉눌림:-3.2% 💰거래대금:342억 RSI:65 기관✅ 외인✅ ⭐ 포스코퓨처엠(003670) 점수:72 🔥급등:+9.5%(오늘) 📉눌림:-1.8% 📁 로그 자동 저장됨
💡14:30에 자동 실행됩니다. 수동으로도 언제든지 실행 가능합니다.
/report
스캔 결과 승률 분석 리포트
과거 스캔 후보 종목들의 실제 결과를 분석해 조건별 통계를 보여줍니다
▶
구문
/report [일수]
예제
/report최근 7일 분석 (기본값)
/report 14최근 14일 분석
/report 30최근 한 달 분석
/update_results
전날 스캔 결과 실제 등락 업데이트
어제 스캔했던 후보 종목들의 실제 다음날 등락을 기록합니다
▶
구문
/update_results [날짜]
예제
/update_results어제 결과 업데이트
/update_results 202605145월 14일 결과 업데이트

종가베팅 — 수동 주문

수동 매수/매도, 잠금 관리
/buy
수동 매수 주문
확인 버튼이 있는 안전한 수동 매수. 가격 0 = 시장가
▶
구문
/buy 종목코드 수량 가격
예제
/buy 005930 10 75000삼성전자 10주 75,000원 지정가
/buy 005930 10 0삼성전자 10주 시장가
/buy 000660 5 180000SK하이닉스 5주 18만원
⚠️확인 버튼을 누르기 전까지 주문이 실행되지 않습니다.
/sell
수동 매도 주문
확인 버튼이 있는 안전한 수동 매도. 가격 0 = 시장가
▶
구문
/sell 종목코드 수량 가격
예제
/sell 005930 10 76000삼성전자 10주 76,000원 지정가 매도
/sell 005930 10 0삼성전자 10주 시장가 즉시 매도
/lock  /unlock
당일 자동 매수 잠금/해제
특정 종목의 당일 자동 매수를 차단합니다. 15:30에 전체 자동 초기화
▶
구문
/lock 종목코드
/unlock 종목코드
예제
/lock 005930삼성전자 오늘 자동 매수 차단
/unlock 005930삼성전자 잠금 해제
